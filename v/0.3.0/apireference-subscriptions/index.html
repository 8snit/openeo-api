openEO API for Subscriptions 0.3.0

The openEO API specification for interoperable cloud-based processing of large Earth observation datasets.

This is a subset of the openEO API, which handles WebSocket-based protocols for subscriptions and notifications.

Publish org.openeo.jobs.output

Message

Data written to the output console via processes.

Payload

Name Title Type Format Default Description
job_id (Required) string Unique identifier of a job that is generated by the back-end during job submission.
issued (Required) string date-time Date and time when the message was sent.
type (Required) string Message type
output (Required) object Output data

Example of payload (generated)

{
  "job_id": "a3cca2b2aa1e3b5b",
  "issued": "2018-08-03T11:53:41Z",
  "type": "org.openeo.jobs.output",
  "output": {}
}

Publish org.openeo.jobs.debug

Message

Debugging information.

Payload

Name Title Type Format Default Description
job_id (Required) string Unique identifier of a job that is generated by the back-end during job submission.
issued (Required) string date-time Date and time when the message was sent.
type (Required) string Message type
debug (Required) object
message (Required) string Debug message
process object Process throwing the debugging message.
name (Required) string Name of the process.
parameters (Required) object Key-Value-Pairs for the parameters of the process. The keys are the parameter names and the values are the actual values specified for the parameter.

Example of payload (generated)

{
  "job_id": "a3cca2b2aa1e3b5b",
  "issued": "2018-08-03T11:53:41Z",
  "type": "org.openeo.jobs.output",
  "debug": {
    "message": "Invalid SRS specified, defaulting to EPSG:4326.",
    "process": {
      "name": "filter_bbox",
      "parameters": {
        "srs": "EPSG:9999",
        "left": 55,
        "right": 56,
        "top": 11,
        "bottom": 10
      }
    }
  }
}

Publish org.openeo.jobs.status

Message

Inform about a status change.

Payload

Name Title Type Format Default Description
job_id (Required) string Unique identifier of a job that is generated by the back-end during job submission.
issued (Required) string date-time Date and time when the message was sent.
type (Required) string Message type
status (Required) string Current status
progress number Progress of a running job in percent.

Example of payload (generated)

{
  "job_id": "a3cca2b2aa1e3b5b",
  "issued": "2018-08-03T11:53:41Z",
  "type": "org.openeo.jobs.output",
  "status": "running",
  "progress": 75.5
}
